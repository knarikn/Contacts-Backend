# Contacts Management API (Node.js / Express)

This is a backend API for managing contacts. It supports user registration, login, and CRUD operations for contacts.  

## Features

- User registration and authentication (JWT)
- Create new contacts
- Retrieve list of user's contacts
- Edit existing contacts
- Each user can only access their own contacts

> Currently uses in-memory storage (no database). Data is lost when server stops.
